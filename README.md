# Prova prática da Supero

Teste prático para vaga de desenvolvimento na Supero TI.

Projeto: Tasklist

O usuário deve poder:

- Adicionar novas tarefas;
- Marcar e desmarcar o status de concluído;
- Editar o conteúdo da task;
- Deletar uma task;
- Frontend da lista responsivo e usabilidade (drag’n’drop);

Usado para rodar:

- Desenvolvido na IDE Eclipse Neon.
- Tomcat 9
- Java 8
- JDK 1.8
- Banco Mysql, usuário root e senha root 
- Frameworks FrontEnd JQuery, Bootstrap
- Spring MVC

Arquivo SQL encontra-se na WebContent/tarefas.sql 

Para o Mysql local, use o MySQL Workbench

Ao instalar no seu localhost acesse pelo endereço: http://localhost:8080/testeSupero
